# PWA Icons

These are placeholder icons for the Recall Progressive Web App.

## Creating Production Icons

For production, generate proper icons using [RealFaviconGenerator](https://realfavicongenerator.net/) with Recall branding:

- Primary color: Cyan (#00d4ff)
- Secondary color: Purple (#9333ea)
- Background: Dark (#0a0e14)

## Required Sizes

- `icon-192.png` - 192x192px
- `icon-512.png` - 512x512px

These sizes are required for PWA installation and app icons on various devices.
